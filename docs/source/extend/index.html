Extending hybrix

hybrix is build to be integrated and build to be extended, making it even more powerfull and better suited for your needs.

What are you going to build?

  • Build applications to connect with hybrix
    • Connect using API a http interface to connect with hybrix nodes.
    • Connect using hybrix-jslib the Javascript library for NodeJS and web.

  • Extend hybrix
    • Add assets
      • Add a token without writing any code. Add a token for chain that is already supported in hybrix.
      • Add new a chain Integrate a new chain to the hybrix platform.
    • Add modules: node side modules integrated with hybrixd to add functionality and/or extend the hybrix API
