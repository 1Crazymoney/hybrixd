Welcome to the hybrix Javascript Library guide

This library can be used to connect software to leverage the capabilities of the hybrix platform. It serves two purposes: first to facilitate the logistics of interfacing with the hybrixd REST API, secondly to handle all client side operations securely and privately. This ensures that keys required for transaction never leave the users device and communication over an encrypted channel.

init {http, https, XMLHttpRequest, custom} Internet of Coins main API interface object. It connects to servers running Internet of Coins Hybridd using the REST API. For reference: [Hybridd API]{@link https://wallet1.internetofcoins.org/api/help}
Parallely executes several steps and collects results in a single object.
data- one of the three methods below must be passed:
data.http- a http object. ({@link https://nodejs.org/api/http.html})
data.https- a https object. ({@link https://nodejs.org/api/https.html})
data.XMLHttpRequest- a XMLHttpRequest object. ({@link https://developer.mozilla.org/nl/docs/Web/API/XMLHttpRequest})

data- Not used
dataCallback- Called when the method is succesful.
errorCallback- Called when an error occurs.
